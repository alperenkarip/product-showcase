# Boilerplate ADR Mirror

Bu dizin `../boilerplate/docs/adr/` altindaki ADR setinin read-only mirror'idir.

Kurallar:

1. Bu dizindeki `ADR-*` ve `18-adr-template.md` dosyalari elle duzenlenmez.
2. Guncelleme yalniz `tooling/sync/upstream-sync.sh` ve `tooling/sync/upstream-sync-manifest.yaml` hattiyla yapilir.
3. Proje-ozel kararlar bu dizine degil `project/adr/` altina `PROJECT-ADR-*` formatiyla yazilir.
